# Calculator App

This is a simple calculator app that performs addition, subtraction, multiplication, and division using the Rhino library.

## Usage

1. Clone the repository.
2. Compile the Java source code.
3. Run the app.

## Features

- Addition
- Subtraction
- Multiplication
- Division
